# Быстрый старт

1. Загрузи весь этот проект в **GitHub**.
2. В **Render** нажми **New -> Blueprint** и выбери этот репозиторий.
3. Render сам прочитает `render.yaml`. Когда дойдёшь до переменных, вставь только **BOT_TOKEN**.
4. Дождись деплоя, открой своего бота и напиши ему `/start` со своего аккаунта **@Dead_Hard11**.
5. После этого у бота появится кнопка открытия приложения, а заявки начнут приходить тебе в Telegram.

## Важно

- Проект настроен под **Starter** на Render, потому что приложению нужен **persistent disk** для базы заявок, отзывов и календаря.
- `PUBLIC_BASE_URL` вручную заполнять не нужно: приложение берёт внешний URL из `RENDER_EXTERNAL_URL` / `RENDER_EXTERNAL_HOSTNAME` автоматически.
- Если потом захочешь поменять название сервиса на Render, код продолжит работать.
